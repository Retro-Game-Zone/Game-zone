Connect Four(2 Player)

How to Play:

  1. Turns:
    • Each player takes turns to drop a disc into one of the columns.
    • The disc will fall down to the lowest empty space in that column.

  2. Winning:
    • The first player to align four of their discs in a row, either horizontally, vertically, or diagonally, wins the game.
    • The win can be:
      • Horizontal
      • Vertical
      • Diagonal

  3. Game End:
    • The game ends when either:
      • One player gets four discs in a row (and wins).
      • The board is completely filled and there are no more moves, resulting in a draw if no one wins.
  4. Draw:
    • If the board is full and no player has connected four discs in a row, the game ends in a draw.
