Memory Game

How to Play:

  1. Find all pairs of matching alphabet cards.

  2. Flip two cards at a time to reveal the letters.

  3. If the cards match, they stay face up.

  4. If they don't match, they flip back over.

  5. The game ends when all pairs are matched.

  6. Your time and moves are tracked.

  7. Try to complete the game in the fewest moves and time possible!
